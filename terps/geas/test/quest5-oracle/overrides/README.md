# Curated winning override scripts

Each `<game>.cmd` here is a hand-authored command script that drives its game to a
genuine ending — or, for the one game that provably has none (WAKE), to the
furthest state the game can reach — for a game whose *raw* walkthrough does not
reach the end as-is —
or (the seven formerly hints-only games) has no linear walkthrough at all, only
Q&A/prose hints from which the script was derived against the game source — or
(The Shack, The Tree, The Zen Garden, Iron John, La Aventura de Parodi, Warriors,
Bear's Epic Quest, and every later addition)
has **no published walkthrough of any kind**, the script being derived entirely
from the game source (its `corpus.tsv` walkthrough column is `-`).
When `overrides/<game>.cmd` exists, `run_corpus.sh` uses it verbatim (including any
title-screen preamble baked into line 1) instead of running `extract_walkthrough.py`
on the walkthrough. Every override starts with a `#`-comment header documenting
exactly how it deviates from the raw walkthrough — the goal is "faithful to the
walkthrough's *intent*, corrected only where the walkthrough or the extractor could
not reach the win."
