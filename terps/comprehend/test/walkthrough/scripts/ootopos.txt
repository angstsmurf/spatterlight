# Oo-Topos -- full winning walkthrough.
#
# A complete deterministic playthrough to the closing "But your part has been
# done! Congratulations!" victory, from the prison cell to the fuelled departure
# from Oo-Topos. The headless terp has no real clock, so this is fully
# reproducible (verified identical across runs) even though Oo-Topos is otherwise
# a pursuit game -- nothing here is rand-gated.
#
# Two things in this script are not literal transcriptions of the prose
# walkthrough on the Classic Adventures Solution Archive, because of how this
# engine's parser resolves the teleporter's magic words:
#
#   * The Oo-Topan teleporter is a one-way pair of rooms linked by two magic
#     words. The writing in the inner room reads "taka ele leva"; the parser
#     resolves "TAKA ELE-LEVA" to the single noun 'taka', which only ever fires
#     the OUTBOUND teleport (room 0x0c -> 0x24). The RETURN teleport (0x24 ->
#     0x0c) is the other magic word, 'leva' (a real dictionary word), so the two
#     return trips below say LEVA, not "TAKA ELE-LEVA". Both teleports also
#     require the light-rod in hand (object 0x0b).
#   * The prose drops the goggles twice; we only carry one pair, so the second
#     drop is omitted.
#
# Win marker (see run_walkthroughs.sh): "But your part has been done!"
# One input per line; blank lines and #-comments are ignored.
