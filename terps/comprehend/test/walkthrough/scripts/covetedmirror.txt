# The Coveted Mirror -- deterministic mid-game script.
#
# Plays from the throne (forced imprisonment) out into the town, collects the
# early items (axe, crystal ball, lockpick, telescope, candlestick, bellows,
# shovel, jug, ladder) and befriends the innkeeper with the grain. Crucially it
# also exercises the HOURGLASS CADENCE: the sand timer (variable 0x11, starts 74,
# -1 per recognised turn) ticks even headless, so the route bribes jailer Boris
# twice at the barrel hub ("Outskirts of town") to stay out of prison -- once
# with the axe, once with the telescope. A bribe resets the sand to 74; the
# barrel scene itself is paused. This makes the script a regression check for the
# throne-room override, the daemon's once-per-turn sand drain, the bribe handler,
# the string banks, the parser and NPC dialogue, all at once.
#
# The riddle at Winder's is fixed in this build (answer UNCLE -- the kinship
# variant; the prose walkthroughs that print MERMAID hit a different random
# variant that does not occur here). The PICTURE/BEAR side-treasure and the
# deeper maze/spell/endgame are deliberately left out -- they need the
# non-deterministic NPC waits and extra bribe detours pinned against MAME first.
#
# Win marker (see run_walkthroughs.sh): "Ah, thou hast made a friend for life!"
# One input per line; blank lines and #-comments are ignored.
