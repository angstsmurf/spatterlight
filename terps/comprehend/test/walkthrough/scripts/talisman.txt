# the ship that leaves the Persian Empire (the boundary between the game's two
# halves). Based on Guido van Otterlo's walkthrough, with the corrections the
# Comprehend port needs:
#
#   * Two BOWs at the start: the first triggers the cell reprieve to King
#     Darius, the second shows respect so he makes his speech and frees you
#     (going anywhere before that gets you executed for disrespect).
#   * The rope and flask are bought by "raising until it closes" -- the
#     merchant's counter erodes toward the offer, so a rising OFFER ladder
#     seals each deal (the source walkthrough's flat 10/13/16 never does).
#   * The shop clerk's order is CP + 65013 (first two letters + 5-digit code),
#     then a card number and its confirmation (rj8906 / ta3756 -- the game does
#     not validate them).
#   * The magic lamp ejects you after four WAITs; only then can you GET LAMP.
#
# Win marker (see run_walkthroughs.sh): "departing the Persian"
