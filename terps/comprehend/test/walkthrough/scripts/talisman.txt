# on Guido van Otterlo's walkthrough, with the corrections the Comprehend port
# needs:
#

#     merchant's counter erodes toward the offer, so a rising OFFER ladder

#     then a card number and its confirmation (rj8906 / ta3756 -- the game does
#     not validate them).
#   * The magic lamp ejects you after four WAITs; only then can you GET LAMP.
#
# Win marker (see run_walkthroughs.sh): "the Empire shalt flourish through eternity"
# One input per line; blank lines and #-comments are ignored.
