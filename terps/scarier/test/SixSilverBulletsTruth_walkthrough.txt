# Six Silver Bullets (William Dooling, 2018) -- the LONG game, "other" ending.
#
# The committed SixSilverBullets_walkthrough.txt takes the SHORT road: burn the
# clock, sneak to the docks, hurl the Silver Dossier into the sea and LEAVE THE
# CITY ("*** You have won ***", blissful ignorance).  This script instead plays
# the game the author actually wrote -- it COMPLETES the assigned mission and
# then, at the dock, READS the dossier to LEARN THE TRUTH: the deep "you have
# lost" ending in which the Silver Agent discovers she is a dying old writer and
# the Gold Agent is the demiurge of the dream.  Same seed (FD_SEED=1234 /
# Scarier erkyrath-1234); full xoshiro conformance with FrankenDrift.
#
# This run also exercises the naming puzzles: it feeds "Thomas Flint" to the
# Black Agent's PopUpInput prompt (scriptable %PopUpInput% -- see a5run_dump's
# popup_from_script hook and FrankenDrift.Headless's TryGetScriptedInput).
#
# Mission (from the yellow note, this seed): KILL THE WHITE AGENT, KILL THE
# INDIGO AGENT, PROTECT THE GREEN AGENT, ASSASSINATE THE GOVERNOR.  Completing

# governor dead) with Suspicion(threatlevel)<=5 lets you WALK RIGHT IN at the
# docks -- the mission-accomplished showdown, not the sneak-through-a-trap one.
#
# Key mechanics threaded here:
#  * The sniper is the Black Agent on the Clocktower (only 4 rooms are exposed).
#    Name him "Thomas Flint" and he trusts you and leaves -- sniper disabled,
#    no bullet spent, the whole city becomes safe to cross.
#  * threatlevel starts at 2.  Sneak past the Widow (keep-walking = +1) to stay
#    at 2; the CEO kill (+1) and the Governor kill (+2) then land it at exactly
#    5, so no "lie low" is needed.
#  * The Indigo Agent only appears (as "the well-dressed gentleman") once
#    threatlevel>=4 AND Time<12, in a street location -- so he is killed last,
#    after the Governor pushes threat to the required level.
#  * The Governor is only reachable in the ballroom once Time<=5 (the party
#    "opens soon"); the Yellow Agent interrupts at nightfall (Time 6) -- nod
#    and leave.  Reaching her needs the ID badge taken from the Skyscraper CEO.
#  * Bullets (6): named Black (0), White (1), the three Enemies of Freedom (3),
#    CEO (1), Governor (0, needs>=1 but does not consume), Indigo (1) = 6.  The
#    dock dilemma is resolved with the bullet-free "keep walking".
