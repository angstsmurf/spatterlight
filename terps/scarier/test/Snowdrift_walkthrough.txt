# Snowdrift (Anonymous, ADRIFT 5.0000353, "Snowdrift V1") -- full winning
# walkthrough, score 50/50, reaching the "To be continued..." ending.  This is a
# short, unfinished demo: you are caught in a blizzard and must find shelter
# before you freeze, then descend into a buried building, solve a phone-dial
# code, and ride an elevator up to where the (never-written) rest of the game
# would begin.  It MATCHES FrankenDrift 0|0 in BOTH RNG modes.
#
# Two mechanics worth knowing before the route:
#
#  * The FREEZE timer (a turn-based event) kills you 15 turns into the storm,
#    with warnings at turns 5 and 10.  You only freeze while OUTDOORS; once you
#    climb down the trap door you are safe.  The route below is 8 commands to
#    get underground, so the timer is never a threat -- but don't dawdle in the
#    snow.
#
#  * The ELEVATOR.  The author built the two elevator doors as *real-time*
#    events (5 seconds, then 56 seconds), and made the tasks that actually open
#    them (ElevatorDo1 / ElevatorDo2) triggerable by the one-character command
#    ".".  In play you press the button and then "wait" -- and because the ADRIFT
#    wait verb "z" is a single character, typing "z" is exactly what fires those
#    tasks and slides the doors open.  So the two "z" lines below are the
#    walkthrough's "wait for the elevator" beats; FrankenDrift opens the doors on
#    the very same inputs (verified 0|0).
