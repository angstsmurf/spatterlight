# Alyas of Starhollow (AoS v.4), by Larry Horsfield -- ADRIFT 5 walkthrough
# regression script.
#
# The game ships an embedded author walkthrough (type WALKTHROUGH or HELP in
# game) as a single ' - '-separated line.  That built-in solution, however, is
# for an OLDER map and DESYNCS on this v.4 build -- replayed verbatim it never
# wins, ending stuck in the starting yort.  This script is a re-derivation that
# plays this build to a MAXIMUM-SCORE "You Have Won!" (650/650 -- the game
# prints its special "scoring the maximum 650 points!" finale line), verified
# byte-exact against FrankenDrift.  Divergences from the embedded walkthrough
# (all confirmed in-engine) include:
#
#   * mill race: "put vegetation in race" (not "push vegetation in water") to
#     block the wheel, lure out the miller and grab a flour sack.
#   * "get book" + "read book" in the roundhouse -- the North@West-Bank mission
#     gate (cl_BookRead) requires the spider book read while held.
#   * "climb up tree" (bare "climb tree"/"climb oak tree" only prompt for a
#     direction); the talking oak needs 1/2/3/4 answered then hands free.
#   * temple: "in" (not "n") into the dark room of deception; follow-string works
#     only after "tie string to column".
#   * inn: choose "1" (buy a round) after 2/3; "g" (again) is unsupported.
#   * Wormwold: keep the tea towel the beggar drops and wrap the farmer's
#     sandwiches in it so hands are free to mount Muffin.
#   * mermaid pond: stow the (undroppable) cloak in the bag before stripping to
#     dive.
#   * castle: the whole bailey/keep/brothel map was rebuilt.  Route to the
#     red-lantern brothel (NE corner), see Room 3, buy sleeping powder at the
#     apothecary (SE corner, gated on having seen Room 3), drug the wine, take
#     the guard uniform.  The keep guards jail you if you carry the travel cloak,
#     which cannot be dropped -- stash it inside the flour sack and leave the
#     sack at the drawbridge, keeping the quest bag.
#   * keep: descend the central well (climb down chain) only AFTER pulling
#     Zagdor's dungeon lever (which reveals the wall brick -> Sigil of Tarkyod);
#     "qwerty" despatches the ambushing thief; give the Sigil to Zagdor and walk
#     him into the King's Audience Chamber to vanquish Xarkzis and recover the
#     Pipe.  The endgame (repair dinghy, cut a mast from the yort frame, rig the
#     bed sheet as a sail, push off) then sails home to win.
#
# The last 30 points (620 -> the maximum 650) are six easily-missed 5-point
# "content find" bonuses, located by dumping every IncVariable-Score task and
# diffing against an A5_DEBUG_SCORE trace of the winning run:
#
#   * "read notice" in the Priests' Dressing Room (no weapons on the balcony).
#   * "x table" while sitting in the Wormwold inn (the Torak graffiti;
#     "read graffiti" is the same 5, guarded by cl_GraffitiSe).
#   * "read sign" at By Paddock -- MUST be read BEFORE knocking on Yosser's
#     door: the sign task teleports Muffin ToSameLocationAs the mule, so read
#     while the mule is still in the paddock (later, with Muffin following and
#     the mule Hidden, it would teleport Muffin into Hidden and strand you).
#   * "rub ring" (the onyx ring from the crow's nest, worn) anywhere except the
#     tree locations 152/158/197 -- an annoyed genie confiscates it, +5.
#   * "read labels" in the Apothecary's Dispensary.
#   * "open case" + "x pipe" after the King's-Audience-Chamber finale hands you
#     the Pipe of Rummkub inside a CLOSED wooden case.
#
# NOT counted in the author's 650 (verified over-max alternates, all still
# reachable): waiting in the flour store with the sack bagged for Dusty's
# return (+5, cl_NoSack2), "rig dinghy" instead of the three manual fit/tie
# steps (25 vs 15, +10), and dodging the -15 cl_NoPlank/NoNails/NoNails2
# penalties by cutting the yort-frame mast BEFORE consuming the repair tools
# (the author's own solution eats those penalties, and the 650 max assumes
# them).  Playing all of them yields 680/650.
#
# The two leading commands drive Horsfield's start-up menu: "c" clears the title
# notice, "b" begins from the first location.
#
# Run with:
#   test/a5_groundtruth.sh test/adrift5-games/"AoS v.4.blorb" test/AoS_walkthrough.txt
