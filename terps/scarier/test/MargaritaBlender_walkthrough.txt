# A self-mocking joke game set in the real Makers Local 256 hackerspace
# (Huntsville, AL): Tim's margarita blender vanished everyone into another
# dimension; rebuild it from shop junk and turn it on.
#
# Entry: the keypad code 80085 (email #2) -- the USB key alternative burns
# up.  The assemble step (at the work table) needs ALL of: clean blender +
# circuit board + motor + flux capacitor ON the table, and the socket set,
# SONIC screwdriver and Dremel held.  So the Dalek easter egg is mandatory
# (wear dalek -> sonic screwdriver) and the Dremel hides under the library
# couch cushion.  Win = turn on contraption while holding the auth token
# (climb member storage with ladder; the ladder is welded from the bike-shed
# scrap; the padlock picks with the lockpick from `look under table`).
#
# HEAT: a repeating 20-turn event overheats you (RANDOM 15-20 subevent) and
# teleports you to the air-conditioned Office; a Tab soda (search couch ->
# 50 cents -> soda machine, repeatable -- the change crawls back into the
# couch) suspends it 20 turns.  This route drinks two Tabs and rides the one
# remaining overheat as its trip to the Office.  The vanilla-FD diff shifts
# on those RANDOM timers; the xoshiro-aligned run is byte-exact.
#
# Data quirks: `enter 330` (Tim's computer password, from scanning the blue
# resistor) NEVER unlocks the computer -- the task carries a second Object
# specific its single-%text% command can never bind, in FrankenDrift too --
# so the schematic is unobtainable flavor and this route skips it.  Several
# tasks (weld/climb/use-screwdriver) check no location and work anywhere.
