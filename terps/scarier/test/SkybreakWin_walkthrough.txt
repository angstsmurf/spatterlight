# Skybreak! by Colton Douglas Olds (ADRIFT 5) -- FULL WINNING playthrough.
#
# Character: Vela, a Human Storyteller+Earthling (Ace, Genius, Skill
# Specialization talents; +2 Acumen, +2 Perception).  The Storyteller ending is
# the one deterministic win this game offers a fixed-seed replay: gather 10
# anecdotes, 10 tales of adventure and 10 horror stories (task SorcererEn2
# fires the "obtain enlightenment" prompt the next time the encounter slot is
# free -- here, on a routine photonics charge -- and "1) Retire" is EndGame
# Win via task Sorcerer1W1).  Unlike the Earthling ending (random jump must
# land at Irth) and the Explorer ending (all ten body types behind rarity
# rolls up to >90), story cargo accrues from encounter successes everywhere,
# so a greedy skill-aware route can bank the 30 stories without any
# unwinnable-trap exposure.
#
# The route: 124 hyperspace jumps through ~89 systems, 28 bespoke encounters
# (Pastries, Ice Fishing, The Demon Brain, The Malacoid Entity, the Psychic
# Vampires survived at hp 1, Gone Mining, The Face of Veth, ...), ending in
# the dreaded Ghooric zone, whose repeatable "decipher the transmissions"
# activity supplies the final horror stories.  Derived with a greedy
# auto-player (seek2.py, kept out of tree -- see TODO_a5_walkthrough_bugs.md)
# that replays test/a5run_dump from scratch per candidate command (0.1s/run,
# fully deterministic), reads state via A5_DUMP_VARS, skips menu options with
# unmet "(You have 0)" requirements, rejects no-op picks, and scores
# candidates with a 3-step greedy rollout on story cargo progress.
#
