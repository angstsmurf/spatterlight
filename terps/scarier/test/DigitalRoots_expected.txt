This game is set in a science fiction world where Sentient Ai, but non-sentient Ai are commonplace.
Mankind has spread throughout the galaxy, some thriving and some dying out.  A young twenty something grand-daughter is missing and Reeves a private investigator, is hired to find her. He manages to track her down, where has has been volunteering for a research foundation. The wealthy grand-parents had obtained credible information from an info-broker. This this data Reeves has been able to learn she is on a Science research vessel called 'Forerunner' somewhere in the Antvas Belt. Upon further investigation within the system itself he learns this ship has very recently crashed on planet Atria Gamma. It has sustained heavy damage but is mostly intact and Reeves is about to go inside to find her dead or alive.

You play as Reeves who must find the missing daughter believed to have been aboard this crashed ship.

If you have never played an Interactive fiction game before, type HELP or you can type COMMANDS to see the list of official and custom commands this game uses.

Basic movement: (All locations or rooms will have listed known exits)
Movement is like going by a cardinal compass to move to a new location you type the direction, like you want to move north you type: north or n

There is a map window which you can see the name of the locations you have visited and the connected paths. However the browser version does not have a map, and the user will have to draw their own map if they want to.

Actions:
I will stick with basic commands you will be using mostly in this game.
You can LOOK to see current room description
EXAMINE an object that might be present, objects you can examine are often listed in the room description by themselves or is written in a descriptive way. However sometimes some objects are mentioned as background objects and the Player character will just glance at it with no description give, you cannot interact with those. Some key items or objects in the game will be able to be examined for details to read.
INVENTORY this lists what the Player character is holding and wearing any other info given.
UNDO this command is handy if you want to take back a command
SAVE this will save your current game and you will be asked to create a save name. See below on how to load a save.
TAKE OBJECT
DROP OBJECT
TALK TO CHARACTER

COMMANDS Lists all the commands that can be used in the game

Loading a Saved Game (In Frankendrift)
Click on GAME at the top left menu and choose Restore a Saved Game. This is only available of the game file is currently loaded. If Restore a Saved Game option is greyed out, this means you FIRST must load the main game file. THEN click on Restore a Saved Game, in Game menu.

Restart a current loaded game (In Frankendrift)
Click on GAME at top, and choose Restart Game or Ctrl + Shift + R

Open Map window (In Frankendrift)
If you accidentally close this window you can get it back again in the GAME menu or Ctrl + M, it has its own window.
